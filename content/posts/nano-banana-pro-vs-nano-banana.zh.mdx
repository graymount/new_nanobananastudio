选择正确的 AI 模型可以极大地影响图像生成效果。Nano Banana Studio 提供两款强大的模型：**Nano Banana**（Gemini 2.5 Flash Image）和 **Nano Banana Pro**（Gemini 3 Pro Image Preview）。以下是你做出正确选择所需了解的一切。

## 快速对比

| 功能 | Nano Banana | Nano Banana Pro |
|-----|-------------|-----------------|
| **底层模型** | Gemini 2.5 Flash Image | Gemini 3 Pro Image Preview |
| **速度** | 快速 | 中等 |
| **最大分辨率** | 1024px | 4096px (4K) |
| **参考图片** | 1 张 | 最多 14 张 |
| **文字渲染** | 基础 | 高级（专业质量） |
| **推理能力** | 标准 | 「深度思考」模式 |
| **最适合** | 快速迭代、草稿 | 专业资产、复杂场景 |
| **积分消耗** | 每次 2 积分 | 每次 2 积分 |

## Nano Banana (Gemini 2.5 Flash Image)

### 优势

**速度冠军**
Nano Banana 针对高吞吐量、低延迟任务进行了优化。如果你需要快速迭代探索想法，这是你的首选。

**性价比高**
与 Pro 相同的积分成本，但更快的生成速度意味着你可以在更短时间内进行更多实验。

**适合草稿**
非常适合在最终高分辨率渲染之前进行初步概念探索。

### 局限性

- 最大 1024px 分辨率
- 编辑时只支持单张输入图
- 基础文字渲染（可能出现拼写问题）
- 标准推理，无「深度思考」模式

### 最佳使用场景

- 快速原型设计和想法探索
- 社交媒体内容（微博、小红书）
- 快速编辑和风格转换
- 批量生成以供挑选

## Nano Banana Pro (Gemini 3 Pro Image Preview)

### 优势

**「深度思考」推理**
Pro 使用高级推理来理解复杂的提示词。它在生成前会考虑物理规律、构图和你的意图——从而产生更连贯的图像。

**4K 分辨率**
生成高达 4096px 的图像，适用于专业印刷和商业用途。

**多图合成**
将最多 14 张参考图片合成为一个连贯的场景。根据 [Google 官方文档](https://ai.google.dev/gemini-api/docs/image-generation)，支持：
- 最多 6 个对象保持一致性
- 最多 5 个人物保持角色一致性
- 复杂场景构图

**卓越的文字渲染**
Pro 擅长渲染清晰、风格化的排版——非常适合海报、标志和信息图。

**Google 搜索接地**
连接实时数据，生成准确的图表、图示和信息可视化。

### 最佳使用场景

- 专业营销材料
- 印刷级艺术作品（海报、周边商品）
- 复杂的多元素场景
- 文字密集的设计（海报、信息图）
- 角色设定图和品牌一致性
- 商业产品摄影

## 真实场景示例

### 场景 1：社交媒体内容创作者
**推荐：Nano Banana**

你需要每天创建小红书帖子，需要快速产出。速度比 4K 分辨率更重要，因为社交平台会压缩图片。

### 场景 2：平面设计师
**推荐：Nano Banana Pro**

你正在设计一张音乐会海报，乐队名称需要醒目展示。Pro 的文字渲染确保排版清晰且拼写正确。

### 场景 3：电商产品图
**推荐：Nano Banana Pro**

你需要将产品照片与生活场景背景合成，同时保持品牌一致性。Pro 的多图功能完美处理这种需求。

### 场景 4：游戏概念美术师
**推荐：两者都用**

使用 Nano Banana 进行快速概念探索（10-20 次快速迭代），然后切换到 Pro 为已批准的概念进行最终高分辨率渲染。

### 场景 5：角色设计
**推荐：Nano Banana Pro**

你需要在多个姿势和场景中保持角色外观一致。Pro 的多图参考（最多 5 个人物）可以保持一致性。

## 何时使用哪个模型

### 选择 Nano Banana 当：
- 你需要几秒内出结果，而不是几分钟
- 你在探索想法，不需要最终质量
- 输出将在屏幕上查看（非印刷）
- 你想快速迭代多个变体

### 选择 Nano Banana Pro 当：
- 你需要印刷级分辨率（4K）
- 你的图像包含必须可读的文字
- 你要合成多张参考图
- 结果用于专业/商业用途
- 你需要复杂的场景构图

## 专业技巧

### 混合工作流
许多专业人士会同时使用两个模型：
1. **草稿阶段：** Nano Banana 进行 10-20 次快速迭代
2. **筛选：** 挑选最佳的 2-3 个概念
3. **精修：** Nano Banana Pro 进行最终高分辨率渲染

### 图像中的文字
如果你的图像包含任何文字：
- **Nano Banana：** 可能出现拼写错误或文字不清晰
- **Nano Banana Pro：** 可靠地正确渲染文字

对于海报、标志或信息图等文字密集的设计，务必使用 Pro。

### 分辨率规划
- **网页/社交媒体：** Nano Banana 的 1024px 足够
- **印刷/商业用途：** Nano Banana Pro 的 4K 是必需的
