想创建令人惊艳的 AI 图像？秘诀不仅在于模型，更在于你如何编写提示词。本指南将 AI 图像生成最佳实践提炼成可立即使用的技巧。

## 核心原则：描述性叙事

最有效的图像生成方法是**描述性叙事**。与其罗列零散的关键词，不如用"叙事性的描述段落"来为场景提供上下文，Nano Banana 的 AI 能更好地理解这种表达方式。

**不要这样写：**
```
狗, 公园, 阳光, 开心, 4k, 杰作
```

**应该这样写：**
```
一只金毛犬在阳光斑驳的公园里玩飞盘。温暖的秋日午后，
狗狗正在跳跃中接住红色飞盘，周围落叶飞舞。
温暖的黄金时段光线。
```

## 提示词框架

使用以下框架来构建提示词，获得稳定的效果：

```
[主体+形容词] 在 [地点] 做 [动作]。
[构图]。[光线]。[风格]。
```

**示例：**
> 一个发着蓝色光芒眼睛的沉稳机器人咖啡师，在未来风格咖啡馆里冲咖啡。广角镜头。温暖的晨光。电影风格。

## 6 种经过验证的提示词策略

### 1. 写实照片风格

使用摄影术语来获得逼真的图像：
- 相机角度（低角度、鸟瞰）
- 镜头类型（85mm 人像镜头、广角）
- 光线条件（黄金时段、影棚灯光）
- 具体的材质和纹理

**示例：**
> 爵士音乐家在昏暗的酒吧里的肖像。85mm 镜头拍摄，浅景深，柔和的背景虚化。来自上方的戏剧性聚光灯，空气中弥漫着烟雾。

### 2. 风格化插画

明确指定艺术风格和视觉特征：

**示例：**
> 吉卜力风格的可爱狐狸角色，柔和的水彩画风格，梦幻的森林背景。清晨的阳光透过树叶洒落。

### 3. 精准的文字渲染

对于标志、海报或信息图，描述字体风格并包含确切的文字内容：

**示例：**
> 一张复古音乐会海报，顶部中央用粗体红色衬线字体写着「JAZZ NIGHT」。装饰艺术风格的几何图案，黑金配色。

### 4. 产品摄影

指定影棚设置以获得商业级图像：

**示例：**
> 专业产品照片：深色大理石台面上的奢华手表。三点式影棚灯光，柔和的反射，45 度角拍摄。干净的白色背景，带有细微阴影。

### 5. 极简构图

强调留白，适合用作文字叠加的背景：

**示例：**
> 极简风景画，广阔的天空占据画面三分之二。地平线上只有一棵小树的剪影。从浅蓝色到暖桃色的柔和渐变。

### 6. 连续叙事

用于连贯的多画面故事：

**示例：**
> 漫画分镜：一个超级英雄在夜间降落在屋顶上。粗线条轮廓，鲜艳的色彩，戏剧性的透视。包含动态线和写着「我来了」的对话框。

## 光线：氛围的关键

光线会极大地影响图像的情感基调：

| 光线类型 | 氛围/效果 |
|---------|----------|
| 黄金时段 | 温暖、怀旧、浪漫 |
| 蓝色时刻 | 神秘、忧郁 |
| 正午强光 | 戏剧性、强烈 |
| 烛光 | 亲密、温馨 |
| 霓虹灯 | 赛博朋克、未来感 |
| 柔光箱 | 干净、专业 |
| 逆光/剪影 | 戏剧性、空灵 |

**始终指定一天中的时间**来确定光线：
> 咖啡馆里女性的肖像。**下午晚些时候的金色阳光**从窗户洒入，投射出长长的影子。

## 常见错误

### 1. 关键词堆砌
**不要：** "美丽、惊艳、4k、杰作、最佳质量、高度精细"
**应该：** 用具体细节描述为什么它美丽。

### 2. 否定式表达
**不要：** "没有汽车、没有行人、没有垃圾的街道"
**应该：** "黎明时分空旷整洁的鹅卵石街道"

### 3. 模糊描述
**不要：** "一个奇幻角色"
**应该：** "身穿银色树叶纹饰板甲的精灵弓箭手，手持反曲弓"

### 4. 忽略构图
**不要：** 只描述主体
**应该：** 包含取景方式："特写肖像"、"远景全景"、"过肩视角"

## 迭代优化

专业技巧：**"不要期望第一次就得到完美的图像。"**

使用多轮对话来优化：

1. **初始提示：** "一个舒适的窗边阅读角"
2. **优化：** "添加更多植物和更温暖的光线"
3. **细节：** "让窗外呈现下雨天的景象"
4. **最后润色：** "在椅子上添加一只睡着的猫"

这种迭代方法通常比一次性指定所有细节效果更好。
